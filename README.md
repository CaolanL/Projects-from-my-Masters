## 2. Intro to Data Science (October 2025)
In assignment 5 of IDS, the goal was to develop both theoretical understanding and practical skills in *reinforcement learning*. We answered questions combining fundamental principles (eg. policies, rewards, and Markov states) with implementing and evaluating a learning agent using a simulated environment. The overall goal is to demonstrate how reinforcement learning algorithms can be designed, trained, and analysed to solve decision-making problems in dynamic environments. \
The files attached include the assignment debrief, our report and all necessary python scripts. 

Also included under the data science projects, is a poster we designed and presented on **SaLSA**. In our pre-assigned groups, we had to pick a paper from a list given, to present a poster on. SaLSA is an improved stochastic line search method designed to automate and stabilise learning rate selection in optimisation algorithms. It addresses the limitations of existing approaches (eg. sensitivity to noise, instability, and high computational overhead) by incorporating exponential smoothing and adaptive update strategies. The goal is to enhance the efficiency and reliability of training modern machine learning models, demonstrating that SaLSa can achieve lower loss, improved stability, and reduced need for manual hyperparameter tuning.

## 3. Iterative Algorithms (January 2026)
In the last computational exercise of Iterative Algorithms, we were tasked with implementing the *Regularised Image Inpainting Problem* for a chosen image. This problem involves repairing damaged pixels for a given image using an iterative solver. On top of some theory on fixed points, we studied how various parameters such as proportion of damaged pixels, affect the iterations for an approximate solution to the problem. \
Files included are the assignment debrief, our report, and the python script written for this problem.
