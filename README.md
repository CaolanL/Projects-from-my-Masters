## 1. CFD - Assignment 5
The aim of this assignment was to first analyse the treatment of div $u^n$ in the explicit time-discretised Navier-Stokes equation 
in a 2-D setting. Secondly, we analysed the effect of various boundary conditions on the velocity profile in a 2-D channel flow, investigating
both inflow and outflow boundaries. The previous assignments required us to implement our discretised equations into Matlab, however
for this last assignment, we were given a N-S solver executable with a UI. \
The files include our report for the assignment, the assignment debrief "pract5.pdf" and the required files for running the applicaiton.
